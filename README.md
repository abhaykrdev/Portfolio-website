# ⚡ Portfolio V1 (React + Vite)

A modern and scalable version of my portfolio built with **React and Vite**, focused on performance, modularity, and clean UI.

---

## ✨ Features

* Component-based architecture
* Fast development with Vite
* Hot Module Replacement (HMR)
* Clean and modern UI
* Easy scalability for future features

---

## 🛠 Tech Stack

* React
* Vite
* JavaScript (ES6+)
* CSS

---

## 📂 Project Structure

```
├── src/
├── public/
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

---

## 🚀 Getting Started

### 1. Install dependencies

```
npm install
```

### 2. Run development server

```
npm run dev
```

### 3. Open in browser

```
http://localhost:5173/
```

---

## 📦 Build for Production

```
npm run build
npm run preview
```
